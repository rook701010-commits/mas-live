MAS LIVE Core Project Overview

「100人でrook7を倒せ！」

1\. プロジェクト概要

プロジェクト名



MAS LIVE Core



コンセプト



MAS LIVE Coreは、



麻雀能力検査「MAS（Mahjong Aptitude Scale）」を発展させた、



視聴者参加型ライブゲームシステムである。



YouTube配信者をBossとし、



視聴者100人が協力して、



麻雀判断問題20問に挑戦する。



2\. プロジェクト目的



本プロジェクトの目的は、



単なる麻雀問題サービスを作ることではない。



目的:



視聴者が主人公になる麻雀エンターテインメント体験を作ること



である。
